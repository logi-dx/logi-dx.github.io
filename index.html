Logi-DX

Welcome to Logi-DX, where we're committed to providing some nice libraries & tools to help improve the developer experience.

The mission is quite simple, we want to help provide a better developer experience, when developing software applications, there's no funny business, no hidden costs, it's all open source & all free to use.

The enterprise users out there will be happy to note that all projects haven't got any weird licensing challenges either. All projects developed by & maintained by Logi-DX require no recognition, no costs, etc. However, it's worth documenting that all software provided as is, without any warranty of any kind.
